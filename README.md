# Murat Levent - Personal Website

A clean, modern, and responsive personal website built to showcase my professional journey, projects, and skills. This website serves as a digital business card and a portfolio for my work.

## Tech Stack

- **Framework**: [Astro](https://astro.build/) v6
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) v4
- **Font**: JetBrains Mono (self-hosted via @fontsource)
- **Theme**: Terminal-inspired dark/light mode

## Features

- 🖥️ Terminal-themed UI with typewriter effects
- 🌗 Dark/Light mode toggle with system preference support
- ⌨️ Full keyboard navigation (arrow keys for tabs)
- ♿ Accessible with ARIA attributes and skip navigation
- 📱 Responsive design with mobile-first approach
- 🔍 SEO optimized with Open Graph and Twitter Card meta tags

## Sections

- **About** — Introduction and current focus
- **Projects** — Private and open-source work
- **Experience** — Professional timeline
- **Community** — FOSDEM attendance and OSS contributions
- **Contact** — Contact information in YAML style

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
